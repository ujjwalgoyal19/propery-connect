## What is Propery Connect?

Propery Connect is a comprehensive digital classified platform built with a modern Turborepo monorepo architecture. It enables users to list, browse, and connect regarding properties for sale or rent.

## Architecture Overview

This project uses a **Turborepo monorepo** with strict separation between frontend and backend:

- **apps/web/** - Next.js 14 (App Router) frontend
- **apps/api/** - Express.js backend API
- **apps/docs/** - Mintlify documentation (you're here!)
- **packages/shared-types/** - Shared Zod schemas and TypeScript types
- **packages/ui/** - Reusable React components

<Note>
  Types and validations are defined **once** in `packages/shared-types` and imported by both apps to
  ensure consistency.
</Note>
